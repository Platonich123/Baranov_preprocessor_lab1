# Лабораторная работа 1
## Тема: Разработка препроцессора

### Описание
В рамках лабораторной работы разработан модуль препроцессора, который очищает исходный код от служебной и избыточной информации.

Модуль выполняет следующие действия:
- удаляет однострочные комментарии `// ...`;
- удаляет многострочные комментарии `/* ... */`;
- удаляет лишние пробелы и табуляции в начале и конце строк;
- сворачивает повторяющиеся пробелы внутри строки;
- удаляет пустые строки;
- выводит сообщения об ошибках при обнаружении незакрытого многострочного комментария;
- выводит сообщения об ошибках при обнаружении недопустимых управляющих символов.

### Состав проекта
- `preprocessor.py` — модуль препроцессора;
- `test.c` — тестовая программа без лишних символов;
- `test_input.c` — тестовая программа с комментариями и лишними пробелами для проверки работы препроцессора;
- `test_cleaned.c` — результат очистки файла `test_input.c`;
- `example_console_output.txt` — пример успешной работы программы;
- `bad_comment_output.txt` — пример сообщения об ошибке при незакрытом комментарии;
- `invalid_char_output.txt` — пример сообщения об ошибке при недопустимых символах.

### Используемый язык
- тестовая программа: **C**;
- язык реализации препроцессора: **Python 3**.

### Запуск программы
```bash
python preprocessor.py test_input.c test_cleaned.c
```

### Пример результата
После обработки файла `test_input.c` формируется очищенный файл `test_cleaned.c`, в котором отсутствуют комментарии, лишние пробелы и пустые строки.
